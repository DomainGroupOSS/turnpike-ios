Creates a new instance of TPRouteRequest with the proper configuration.

Parameters
urlSchemaThe URL schema of the incoming URL. If invoked internally, this should be nil.
queryParametersThe query parameters parsed from the query string into NSStrings in an NSDictionary, and if invoked internally, this should be nil.
matchedRouteWhich defined route (in route format, ie `user/:user_id') matched the incoming route. If no match was found, the default route will be invoked and this should be nil.
routeParametersAn NSDictionary of route parameters found in the matched route. If no route parameters are found this will be an empty NSDictionary, and if no matched route was found, this should be nil.
Returns
A new instance of TPRouteRequest.
